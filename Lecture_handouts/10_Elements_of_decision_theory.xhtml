1 Why do we need decision theory in Bayesian Stats?

Another important purpose of statistical analysis, apart from qualifying uncertainty via posterior probability distributions, is to make inference about unknown quantities (e.g., parameters, future outcomes, etc...). In the Bayesian framework, the derivation of inferential tools (such as, estimators, hypothesis tests, etc...) is addressed as a decision theory problem, aka we get decisions about the unknown quantities.
A decision is certainly a function of our knowledge quantified by probabilities. For example, a doctor deciding on the best treatment for a patient will certainly use their knowledge of what illness the patient has. Different illnesses may have different probabilities based on the symptoms, test results, and so on, and it would be foolish to concentrate on the most improbable illness and treat for that. Such questions constitute a decision theory problems.
In statistics, the overall purpose of most inferential studies is to provide the statistician (or a client) with a decision (E.g, what estimator to use? Shall I reject a hypothesis?), it seems reasonable to ask for an evaluation criterion of decision procedures that assesses the consequences of each decision and depends on the parameters of the model, i.e., the true state of the world (or of Nature). As we will see such a criterion is the so called loss function (or more rigorously the utility function) introduced in what follows.

2 Set-up of the decision problem (intro)

Definition 1. The decision problem ( Θ , D , ℓ ) involves the following basic elements
  • Decision space: D ={d}
    The decision maker wishes to select a single decision d ∈ D from a space of all possible decisions D .
  • State space (Space of the word or Parameter space): Θ ={ θ }
    The decision process is assumed to be affected by the unknown (uncertain) quantity θ ∈ Θ which signifies the state of the world. The set of all possible states of the world is denoted by Θ . The decision maker perceives that a particular decision d ∈ D results in a corresponding state θ ∈ Θ .
  • Loss function ℓ : Θ × D → R +
    The decision maker assigns a loss function (or error) ℓ ( d, θ ) that evaluates the penalty (or error, or suffer, or regret, or etc...) associated with decision d when the parameter takes the value θ .

Definition 2. Statistical decision problem is a decision problem ( Θ , D , ℓ ) coupled with an experiment e ∈ E ( E denotes the family of experiments) that involves an observable y ∈ Y ( Y denotes the sample space) whose sampling distribution F( y| θ ) depends on the state θ ∈ Θ chosen by nature.

Note 3. The main aim in a statistical decision problem ( Θ , D , ℓ ) is for the decision maker to choose the/an optimal decision d * := d * ( y ) ∈ D .

Example 4. Statistical inference consists of taking a decision d=d( y ) ∈ D elated to the parameter θ ∈ Θ based on the observation y ∈ Y , where y and θ are related by the sampling distribution F( y| θ ) . In many cases, the decision d ∈ D will be a procedure performing: Point estimation (what type of estimator to use?), Credible regions (how to find the bounds?), Hypothesis tests (how to reject the hypothesis?), model selection (which model is the `best'?).
Definition 5. Decision rule is a function which maps the observables y ∈ Y to an appropriate decision d ∈ D . It is defined as δ : Y → D , where δ ( y ) ∈ D , and it is such that ∫ Y ℓ ( θ , δ ( y ) ) d F( y| θ ) < ∞

Note 6. Briefly and more rigorously speaking about statistical decision problems,
  • The decision maker assigns a utility function u: E × Y × D × Θ → R with u( e,y,d, θ ) which describes the value to perform a particular experiment e ∈ E , observing a particular observation y ∈ Y , taking a particular decision d ∈ D , and then finding that a particular θ ∈ Θ obtains.
  • The existence of the utility function u is introduced in Decision Theory 3, and ignored in this handout.
  • Once the utility function u is defined, the loss function ℓ can be specified as ℓ ( e,y,d, θ ) =u( e * ,y, d * , θ ) -u( e,y,d, θ ) ,  or  ℓ ( e,y,d, θ ) =-u( e,y,d, θ ) where e * and d * are optimal choices of the experiment and decision.
  • In this handout, we assume the simplified case where the experiment e is fixed and hence we drop e,y from arguments in u and ℓ ; hence ℓ ( d, θ ) =u( d * , θ ) -u( d, θ ) ,  or  ℓ ( d, θ ) =-u( d, θ ) .

3 Finding Bayesian optimal decision rules (intro)

Question 7. Given a statistical decision problem ( Θ , D , ℓ ) , what is the optimal decision rule δ ( y ) ∈ D ?

Note 8. To get an optimal decision there is a need to derive an effective comparison criterion that orders the possible decisions d ∈ D based on the loss function ℓ ( ⋅ , ⋅ ) , (equiv. the utility function u( ⋅ , ⋅ ) ).

Definition 9. (Frequentist) Risk function R( θ , δ ) :=R( θ , δ ( y ) ) is defined as R( θ , δ ) = E F ( ℓ ( θ , δ ( y ) ) | θ ) = ∫ Y ℓ ( θ , δ ( y ) ) d F( y| θ ) , (1) where δ ( ⋅ ) is the decision rule, i.e, the allocation of a decision to each outcome y∼F( y| θ ) from the random experiment e .

Note 10. For the derivation of inferential tools for θ , such as estimators θ ˆ , the frequentist approach relies on comparisons based on the risk function (1) in order to choose the `optimal' decision θ ˆ = δ ( FR ) ( y ) . The objections are the following:
  • Risk function R( θ , δ ) averages loss function ℓ ( θ , δ ( y ) ) over the different values of y proportionally to the density f( y| θ ) . Therefore, it seems that the observation y is not taken into account any further. It evaluates procedures on their long-run performance and not directly for the given observation y . To average over all possible values of y , when we know the observed value of y , on one hand may lead to interesting mathematical properties, but on the other hand it is rather a waste of information.
  • The frequentist approach implicitly assumes that this problem will be met again and again for the frequency evaluation to make sense. However, there are objections against the this sense of repeatability of experiments. Eg, if new observations become available and one wants to make use of them, this could possibly modify the way the experiment is conducted.
  • Risk function R( θ , δ ) is a function of the parameter θ . There may not exist, in general, an optimal procedure δ ( F ) ( ⋅ ) that uniformly minimises R( θ , δ ) for all θ ∈ Θ . The frequentist approach does not induce a total ordering on the set of procedures. For this reason, frequentist approach, requires additional restrictions (in a rather artificial manner), eg, admissibility.

Note 11. To address the decision problem ( Θ , D , ℓ ) in the Bayesian paradigm the decision maker assigns a probability distribution P( θ ,y ) on the possibility space Θ × Y . Recall (Handout 3; Remark rem:HO3_fghfhfasguarsggfsg) that the joint distribution P( θ ,y ) determines probability distributions such as d P( y, θ ) = d F( y| θ ) d Π ( θ ) = d Π ( θ |y ) d F( y ) where Π ( θ ) is the prior distri., Π ( θ |y ) is the posterior distr., and F( y ) is the prior predictive distr.

Note 12. To derive a quantity able to order decisions in the Bayesian paradigm, it is reasonable to integrate out the loss ℓ ( θ ,d ) on the space Θ , (instead of integrating on the space Y ) and condition on the observables y . This is because θ is unknown and y is known/observed.

Definition 13. The posterior expected loss of a decision d ∈ D when the posterior distribution is Π ( θ |y ) is defined as ϱ ( π ,d|y ) = E Π ( ℓ ( θ ,d ) |y ) = ∫ Θ ℓ ( θ ,d ) d Π ( θ |y ) (2)

Note 14. Posterior expected loss ϱ ( π ,d|y ) integrates the loss ℓ ( θ ,d ) with respect to the posterior distribution Π ( θ |y ) of the parameter θ , conditionally on the observed value y . It is a function of y but this dependence is not troublesome, because y is known (as it is observed).

Note 15. It is reasonable for a Bayesian Statistician to consider as optimal decision the one that minimizes (2) for a given observable y .

Definition 16. The Bayes risk of a decision d ∈ D , with respect to a prior Π ( θ ) on Θ is defined as r( π , δ ) = E Π ( R( θ , δ ) ) = ∫ Θ ∫ Y ℓ ( θ , δ ( y ) ) d F( y| θ ) d Π ( θ ) ; (3) It is the risk function R( θ , δ ) integrated over θ with respect to the prior distribution d Π ( θ )

Remark 17. Bayes risk r( π , δ ) induces a total ordering on the set of potential decisions, and hence it allows for the direct comparison of decisions unlike the risk function R( θ , δ ) . This is because r( π , δ ) associates a real number with every decision d , it is not a function of θ .

Note 18. In the Bayesian framework an optimal decision for the decision problem ( Θ , D , ℓ ) can be found in two ways:
  1. the extensive form of analysis: it minimizes the posterior expected loss ϱ ( π ,d|y ) (2) w.r.t. d min d ∫ Θ ℓ ( θ ,d ) d Π ( θ |y )
  2. the Normal form of analysis: it minimizes the Bayesian risk r( π , δ ) (3) w.r.t. δ min δ ∫ Θ ∫ Y ℓ ( θ , δ ( y ) ) d F( y| θ ) d Π ( θ )

Definition 21. A Bayes rule associated with a prior distribution Π ( θ ) and a loss function ℓ ( ⋅ , ⋅ ) is any δ π which minimizes r( π , δ ) . For every y ∈ Y , it is given by δ π ( y ) such that δ π ( y ) = arg min d ϱ ( π ,d|y )

Definition 22. The Minimum Bayes risk is defined as the value r( π ) =r( π , δ π ) , where δ π is a Bayes rule.

Remark 23. Operationally, to find the Bayes rule δ π ( y ) , we will mainly minimize the posterior expected loss ϱ ( π ,d|y ) .

Remark 24. From a strictly Bayesian point of view, only the posterior expected loss ϱ ( π ,d|y ) is important, because the Bayesian paradigm is based on the conditional approach (conditional on the observations).

Remark 25. The `reasonable' way to view the situation is that minimizing the posterior expected loss ϱ ( π ,d|y ) . One should condition on what is known (aka the observables y ) and integrate/average out on what is unknown (aka θ ). Minimizing r( π , δ π ) that integrates out y seems bizarre from this perspective.

Remark 26. Unlike the Frequentist one, the Bayesian approach is sufficiently reductive to reach an effective decision δ π because, by minimizing the posterior expected risk ϱ ( π ,d|y ) (w.r.t. δ π ), it minimizes the integrated risk r( π , δ π ) (w.r.t. δ π ) which in fact allows a direct comparison of estimators.

Remark 27. The Bayesian approach works conditional upon the actual observation y , as well as it incorporates the probabilistic properties of the sampling distribution F( y| θ ) . This is in contrast to the frequentist approach (Note 10) where the observed information y seems to be wasted because it averages over all possible values of y instead of conditioning on the given observed y .

Definition 28. Generalized Bayes rule is called any δ π ( y ) that minimizes r( π , δ ) or ϱ ( π ,d|y ) for each y ∈ Y where f( y ) >0 but the prior distribution Π ( θ ) is improper.

Remark 29. Definition 21, and Theorem 20 are valid for proper and improper priors, provided that r( π ) < ∞ , otherwise other treatments exist.

4 Admissibility

Contrary to the Bayesian approach, the frequentist approach is not reductive enough to lead to a single optimal rule/estimator, and for this reason frequentist use additional optimality concepts such as admissibility. Here, we study Bayes rules/estimators with respect to the frequentist optimality criterion of admissibility.
Definition 30. A decision rule δ 0 is inadmissible if there exists a decision rule δ 1 which dominates δ 0 , namely: R( θ , δ 0 ) ≥ R( θ , δ 1 ) , ∀ θ ∈ Θ R( θ 0 , δ 0 ) >R( θ 0 , δ 1 ) , ∃ θ 0 ∈ Θ Otherwise, δ 0 is said to be admissible.
Remark 31. Well, I wouldn't use an inadmissible estimator δ because a decision rule with a lower risk could be found...

Admissibility in Bayesian rules

Note 32. One may expect Bayes rule to be admissible because if a rule with better risk R( θ , δ ) existed, that rule would also have better Bayes risk r( π , δ ) = E Π ( R( θ , δ ) ) . This is always true when the prior π is proper, and we study this case in what follows. Surprisingly, however, this is not always true when π is improper (non-informative).
Theorem 33. If the Bayes rule associated with a prior π is unique, it is admissible.
Proof. Let δ π be a Bayes rule. Assume that δ π is inadmissible. So let δ * be any decision rule with R( θ , δ * ) ≤ R( θ , δ π ) , for all θ ∈ Θ . Then r( π , δ π ) -r( π , δ * ) = ∫ Θ R( θ , δ * ) d Π ( θ ) - ∫ Θ R( θ , δ π ) d Π ( θ ) = ∫ Θ ( R( θ , δ * ) -R( θ , δ π ) ) d Π ( θ ) ≤ 0 and so δ * is also Bayes. Because δ π is unique Bayes by assumption, we must have δ * ( y ) = δ π ( y ) for all y ∈ Y . Therefore, δ π must be admissible.

Admissibility in Generalised Bayesian rules

Generalized Bayes rules (Definition 28) may be or may not be admissible.
Remark 36. When the loss is positive and r( π , δ π ) = ∫ Θ R( θ , δ π ) d Π ( θ ) < ∞ , the generalized Bayes rule δ π (improper prior) can be easily shown to be admissible, similar to the Bayes rule case (proper prior; Theorems 34 & 35). In fact in this case, as mentioned in Remark 29, δ π minimizes r( π , δ ) or ϱ ( π ,d|x ) and it can be shown (similar to the Bayes rule) that generalized Bayes rule must be admissible under suitable conditions.

Long run analysis of the behavior of Generalised Bayesian estimator δ π .
Consider the case that an objective Bayesian uses non-informative priors (improper in this case) automatically on a routine bases because he/she does not want to contaminate his/her statistical analysis with subjective elements. Due to this repeated use, the Bayesian enters into the frequentist domain; hence it is reasonable to investigate how repeated use of this prior actually performs.
Consider
  • a sequence of independent problems ( ( θ ( 1 ) , y ( 1 ) ) ,( θ ( 2 ) , y ( 2 ) ) ,... )
