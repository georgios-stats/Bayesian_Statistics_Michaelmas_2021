1 Set-up of a hypothesis test

Aim:
Let y=( y 1 ,..., y n ) generated from the real unknown data-generating process y ∼ R( y ) . Statistician approximates/parametrizes R( ⋅ ) by a statistical model F( y| θ ) with unknown θ ∈ Θ . Then you wish to find useful statements about θ : E.g. is there a smaller Θ ★ ⊆ Θ where You can restrict the possible values of unknown θ ?

Notation 1. Let y=( y 1 ,..., y n ) be a sequence of observables modeled to have been generated from the sampling distribution F( y| θ ) labeled by an unknown parameter θ ∈ Θ following a priori distribution Π ( θ ) ; namely { y| θ ∼ F( y| θ ) θ ∼ Π ( θ ) (1) Assume there is interest to test/compare the following hypotheses/statements H 0 : θ ∈ Θ 0 ; vs. H 1 : θ ∈ Θ 1 (2) where Θ = Θ 0 ∪ Θ 1 , under the Bayesian model (1).
Note 2. The pair of hypotheses (2) partitions the overall prior Π ( θ ) (representing overall prior believes about θ ) as d Π ( θ ) = π 0 × d Π 0 ( θ ) + π 1 × d Π 1 ( θ ) (3) where π 0 , and π 1 describe the prior probabilities of hypotheses H 0 and H 1 π 0 = Pr Π ( θ ∈ Θ 0 ) ⏟ = Pr Π ( H 0 ) = ∫ 1 ( θ ∈ Θ 0 ) d Π ( θ ) , π 1 = Pr Π ( θ ∈ Θ 1 ) ⏟ = Pr Π ( H 1 ) = ∫ 1 ( θ ∈ Θ 1 ) d Π ( θ ) , respectively while Π 0 ( θ ) := Π ( θ | θ ∈ Θ 0 ) and Π 1 ( θ ) := Π ( θ | θ ∈ Θ 1 ) are prior distributions with pdf/pmf π 0 ( θ ) := π ( θ | θ ∈ Θ 0 ) ⏟ = π ( θ | H 0 ) = π ( θ ) 1 ( θ ∈ Θ 0 ) ∫ 1 ( θ ∈ Θ 0 ) d Π ( θ ) ; and π 1 ( θ ) := π ( θ | θ ∈ Θ 1 ) ⏟ = π ( θ | H 1 ) = π ( θ ) 1 ( θ ∈ Θ 1 ) ∫ 1 ( θ ∈ Θ 1 ) d Π ( θ ) ,
describing how the prior mass of θ is spread out over the hypotheses H 0 and H 1 respectively. Then the Bayesian hypothesis test can also be expressed as H 0 :{ y| θ ∼ F( y| θ ) θ ∼ Π 0 ( θ ) , θ ∈ Θ 0 vs H 1 :{ y| θ ∼ F( y| θ ) θ ∼ Π 1 ( θ ) , θ ∈ Θ 1 (4) with prior π 0 = Pr Π ( θ ∈ Θ 0 ) and π 1 = Pr Π ( θ ∈ Θ 1 ) .
Question 3.  Which Bayesian model ( H 0 or H 1 ) describes `better' the real data generating process?
Note 4. In Bayesian framework, hypothesis testing is rather straightforward. All You need to do is to calculate the corresponding posterior probabilities Pr Π ( θ ∈ Θ 0 |y ) , and Pr Π ( θ ∈ Θ 1 |y ) , and decide between H 0 and H 1 .

4 Special cases in hypotheses tests

Definition 16. Traditionally, hypotheses, H j , are categorized as:
  • Single (or point) hypothesis for θ is called the hypothesis H j : θ ∈ Θ j where Θ j ={ θ j } contains a single element, namely when Π j ( θ ) assigns probability one to a specific value for θ .
  • Composite hypothesis for θ is called the hypothesis H j : θ ∈ Θ j where Θ j ⊆ Θ contains many elements. Namely when Π j ( θ ) defines a non-degenerate density π j ( θ ) over Θ j ⊆ Θ .
  • General alternative hypothesis for θ is called the composite hypothesis H 1 : θ ∈ Θ 1 where Θ 1 = Θ -{ θ 0 } and θ 0 a single value. It is often denoted as H 1 : θ ≠ θ 0 and compared against a single null hypothesis H 0 : θ = θ 0 .
We present some special cases of hypothesis tests.
  1. Composite vs Composite is the hypothesis test: H 0 : θ ∈ Θ 0 vs H 1 : θ ∈ Θ 1 where both Θ 0 ⊆ Θ and Θ 1 ⊆ Θ contain more than one elements. Overall prior can be partitioned as d Π ( θ ) = π 0 × d Π 0 ( θ ) + π 1 × d Π 1 ( θ ) Then the conditional marginal likelihoods are f 0 ( y ) = ∫ f( y| θ ) d Π 0 ( θ ) ; f 1 ( y ) = ∫ f( y| θ ) d Π 1 ( θ ) .
