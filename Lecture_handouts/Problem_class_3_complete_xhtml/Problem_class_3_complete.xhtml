2 Inference under model uncertainty

Exercise 2. ( ★ ★ )
Let B k,j ( y ) be the Bayes factor of model M k against model M j , for all ∀ k,i,j ∈ K . . Show that B k,j ( y ) = B k,i ( y ) B i,j ( y ) , for all ∀ k,i,j ∈ K .
Solution. It is B k,j ( y ) = π ( M k |y ) / π ( M k ) π ( M j |y ) / π ( M j ) = π ( M k |y ) / π ( M k ) π ( M i |y ) / π ( M i ) π ( M i |y ) / π ( M i ) π ( M j |y ) / π ( M j ) = B k,i ( y ) B i,j ( y )
