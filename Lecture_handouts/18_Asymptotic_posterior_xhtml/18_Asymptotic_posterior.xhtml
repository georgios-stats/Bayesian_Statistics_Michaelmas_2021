We examine the properties of the posterior distribution Π ( θ |y ) , under different sets of conditions, as the number of observations n increases n → ∞ , as well as their implications in inference.

What is about?

Notation 1. Consider the Bayesian model ( F( x 1:n | θ ) , Π ( θ ) ) as { x 1:n | θ ∼ F( ⋅ | θ ) θ ∼ Π ( ⋅ ) (1) where a sequence of observables x 1:n =( x 1 ,..., x n ) are drawn from the parametric model F( ⋅ | θ ) admitting a pdf/pmf f( ⋅ | θ ) with unknown parameter θ ∈ Θ . The prior Π ( ⋅ ) of θ admits pdf/pmf π ( ⋅ ) .

Question 2. We study the behavior of the posterior distribution Π ( θ | x 1:n ) with respect to the number of observables n , first when θ is a discrete parameter, and then when θ is a continuous one.

Note 3. All the theorems in this chapter are frequentist in character, namely we study the posterior laws under the assumption that the observables x 1:n is a random sample from the sampling distribution F( ⋅ | θ * ) for some fixed non-random true value θ * ∈ Θ .

Notation 4. We denote the likelihood function as L n ( θ ) :=f( x 1:n | θ ) and the posterior distribution Π n ( θ ) with pdf/pmf π n ( θ ) := π ( θ | x 1:n ) ∝ L n ( θ ) π ( θ ) , to easy the notation and make clear Note 3.

1 Discrete θ : Asymptotic consistency

Note 5. Given the Bayesian model (1), we consider cases where θ ∈ Θ is a discrete parameter, and Θ is a countable space.

Note 6. The theorem below implies that, if Θ is countable, under conditions, the posterior distribution function of θ ∈ Θ ultimately degenerates to a step function with a single (unit) step at θ = θ * , where θ * is the true value of the unknown discrete parameter θ .

Notation 17. Let U n ( θ ) = log ( π n ( θ ) ) . Let | θ |= θ ⊤ θ . Let B δ ( θ * ) ={ θ ∈ Θ ;| θ - θ * |< δ } .

4 Continuous θ : Asymptotic efficiency of Bayes Estimates

Remark 30. Consider the squared error loss ℓ ( θ , δ ) =( θ - δ ) ⊤ ( θ - δ ) which implies the posterior expectation δ π = E Π ( θ | x 1:n ) as Bayes point estimator. Given that we can interchange the limit and the expectation operator of ϑ = n ( θ - θ ˆ n ) , we get n ( δ π - θ ˆ n ) → p 0 meaning that δ π and θ ˆ n are asymptotically equivalent; i.e. δ π - θ ˆ n → p 0. Hence n ( δ π - θ * ) = n ( δ π - θ ˆ n ) + n ( θ ˆ n - θ * ) → D N (0,I( θ * ) -1 ) which means that the Bayes estimator is asymptotically efficient.
Example 31. (Cont. Example 29) How the Bayes estimators under the square loss and under the 0-1 loss behave as n → ∞ ?
Solution. The exact posterior distr. is θ | x 1:n ∼ Be ( a+n x ‾ ,b+n-n x ‾ ) . The squared loss and the 0-1 loss imply the posterior mean δ 1 ( x 1:n ) = n x ‾ +a n+a+b and posterior mode δ 2 ( x 1:n ) = n x ‾ +a-1 n+a+b-2 as Bayes estimators correspondingly. Both converge to the MLE θ ˆ n = x ‾ since lim n → ∞ δ 1 ( x 1:n ) = x ‾ and lim n → ∞ δ 2 ( x 1:n ) = x ‾ .

Definition 32. (Types of converge) Assume a probability triplet { Ω , F ,P} , and a sequence of random quantities { x n ;n=1,2,...} , such that x n : Ω → R d , d>0 . Then
  • { x n } converges almost surely to a random quantify x if and only if P( lim n → ∞ x n =x ) =1. It is demoted as x n → a.s. x .
  • { x n } converges in distribution to a random quantify x if and only if for all ε >0 , lim n → ∞ F x n ( t ) = F x ( t ) for all continuity points t of F ⋅ in R , where F x n ( t ) =P( x n ≤ t ) and F x ( t ) =P( x ≤ t ) are CDFs of x n and x . It is demoted as x n →Dx .
