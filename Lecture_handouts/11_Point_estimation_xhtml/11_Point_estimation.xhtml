1 Set-up and aim

Notation 1. Consider a Bayesian model { y| θ ∼ F( y| θ ) θ ∼ Π ( ⋅ ) where y:=( y 1 ,..., y n ) ∈ Y is a sequence of observables, assumed to be generated from the parametric sampling distribution F( y| θ ) with pdf/pmf f( y| θ ) and labeled by an unknown parameter θ ∈ Θ following a prior distribution Π ( θ ) with pdf/pmf π ( θ ) .

The AIM, in parametric (or predictive) point estimation,
is to derive a parametric (or predictive) point estimator δ ˆ ( y ) , a quantity summarizing Your believes about the unknown parameter θ (or unknown future outcome sequence z:=( y n+1 ,..., y n+m ) ) or any function of it in an appropriate manner.
Point estimation is addressed
in the statistical decision theory framework, where the decision rule is the estimator δ ˆ ( y ) , (the decision space results consequently) and the loss function ℓ ( ⋅ , ⋅ ) is set in a subjective manner as a penalty and according to what You may loss or how You may suffer if you use the produced estimator.
Note 2. Although we define the parametric and predictive point estimation as two distinct concepts, it will become clear in Section 4 they are treated in a similar manner.

3 Predictive point estimation

Note 9. The Bayesian point predictive estimator and its standard error are defined similar to the parametric ones.

Note 10. Degree of believe about a future sequence of outcomes z=( y n+1 ,..., y n+m ) ∈ Z is quantified via the predictive distribution d G( z|y ) =g( z|y ) d z with cdf G( z|y ) and pdf/pmf g( z|y ) .

Definition 11. Bayes predictive point estimator of z=( y n+1 ,..., y n+m ) ∈ Z under the loss function ℓ ( z, δ ) and predictive distribution G( z|y ) is the decision rule δ ∈ D = Z which minimizes E G ( ℓ ( z, δ ) |y ) ; i.e. δ ˆ = arg min ∀ δ E G ( ℓ ( z, δ ) |y ) = arg min ∀ δ ∈ D ∫ Z ℓ ( y, δ ) d G( z|y )

Note 12. The accuracy of the predictive point estimator is traditionally presented by using the squared mean (predictive) square error.

Definition 13. Let δ ˆ =( δ ˆ 1 ,..., δ ˆ m ) be the Bayes point predictive estimator of z ∈ Z ⊆ R m with predictive distribution G( z|y ) . Then the standard error of the j -th dimension of δ ˆ is defined as se G ( δ ˆ j |y ) = [ mse Π ( δ ˆ |y ) ] j,j where mse G ( δ ˆ |y ) = E G ( ( z- δ ˆ ) (z- δ ˆ ) ⊤ |y ) is the mean squared error of δ ˆ .

Note 14. A number of standard Bayesian point estimators, along with the corresponding loss functions are examined below. These point estimators correspond to summary statistics of the posterior/predictive distribution (mean, median, mode, quantiles, etc.).

Note 15. Bayesian point estimators are applied to both parametric and predictive inference likewise and hence presented together in what follows.
Notation 16. Consider unknown random quantity x ∈ X ⊆ R k following a distribution Q( x|y ) such as d Q( x|y ) =q( x|y ) d x with cdf Q( x|y ) and pdf/pmf q( x|y ) . These are dummy quantities for the following:
  • In parametric inference, we have x ≡ θ , Q ≡ Π , q ≡ π , and k=d .
  • In predictive inference, we have x ≡ z , Q ≡ G , q ≡ g , and k=m .
  • In more extreme cases, we have x ≡ ( z, θ ) , Q ≡ P , q ≡ p , and k=d+m .
  • Note that x can also be any function of θ or z , or ( z, θ ) ...

Proposition 26. The Bayes estimate δ ˆ of x under the absolute loss function ℓ ( x, δ ) = ∥ x- δ ∥ 1 is the median of distribution Q( x|y ) δ ˆ = median Q ( x|y ) . (3)

Remark 27. The linear loss function in Proposition 24, allows the adjustment of the penalty between over-estimating and under-estimating x , by adjusting c 1 and c 2 . Remark 28. Compared to the linear loss functions ℓ ( x, δ ) = ∥ x- δ ∥ 1 , the quadratic loss ℓ ( x, δ ) = ∥ x- δ ∥ 2 2 aims at over-penalizing large but unlikely errors. The linear loss functions increase much slower than the quadratic loss, and hence, while remaining convex, they do not penalize so much the large but unlikely errors.

Note 31. Consider that parametric inference about θ is of interest. In the frequentist sense, MAP estimator can be seen as a penalized maximum likelihood estimator in the sense that it essentially maximizes log ( π ( θ |y ) ) = log f( y| θ ) + log π ( θ ) - [ normalising const. ] ∝ log f( y| θ ) + log π ( θ ) (6) which is the log-likelihood penalized by the log-prior (!!!)
Example:
Consider the Bayesian model { y| μ ∼ N ( μ , σ 2 ) μ ∼ N ( μ 0 , σ 0 2 ) where σ 2 , μ 0 , σ 0 2 are known. Then (6) becomes log ( π ( μ |y ) ) ∝ log ( N ( y| μ , σ 2 ) ) - 1 2 σ 0 2 ( μ - μ 0 ) 2 Here maximization of the posterior pdf is a compromise between the maximization of the log-likelihood and minimization of the distance ( μ - μ 0 ) 2 regulated by σ 0 2 . Hence the prior shrinks (or penalizes) μ towards μ 0 and this shrinkage is adjusted via σ 0 2 .
Example:
Consider the Bayesian regression model { y| β ∼ N ( Φ β ,I σ 2 ) β ∼ N ( [ 0,...,0 ] ⊤ , diag ( v 1 ,..., v d ) ) where σ 2 , v 1 ,..., v d are known. Then (6) becomes log ( π ( β |y ) ) ∝ log ( N ( y| Φ β ,I σ 2 ) ) - 1 2 ∑ j=1 d 1 v j ( β j -0 ) 2 Here, the prior shrinks all β j towards 0 and this shrinkage is adjusted via v j . Hence it can be speculated that the elements of the MAP estimate vector β ˆ at which the likelihood part would assign them small values (in absolute sense) will end up with much smaller values and closer to zero. This has applications to dimension reduction / variable selection / compressive sensing in specific high-dimensional regression problems.

Remark 32. Zero-one loss imposes a quite forceful penalization; because the penalty is equal to 0 if δ is the correct answer, and 1 if it is wrong.
