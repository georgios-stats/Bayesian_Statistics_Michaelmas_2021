1 Finite mixture distributions

Definition 1. Let P m ={ Π l ( θ | χ l ) ;l=1,...,m} be a collection of probability distributions where { χ l } l=1 m are parameters of the l -th component Π l ( θ | χ l ) . Let { ϖ l } l=1 m be a set of weights where ϖ l >0 and ∑ l=1 m ϖ l =1 . The mixture distribution derived from the aforementioned collections is Π ( θ | ϖ , χ ) = ∑ l=1 m ϖ l Π l ( θ | χ l ) , θ ∈ Θ (1) where χ :=( χ l ,l=1:m ) and ϖ :=( ϖ l ,l=1,...,m ) . Π l ( θ | χ l ) is called l -th mixture component with mixture weight ϖ l .

Example 2. Let h( ⋅ ) be a function defined on Θ . The expectation of h( ⋅ ) with respect to (1) is E Π ( h( θ ) | ϖ , χ ) = ∫ ∑ l=1 m ϖ l h( θ ) d Π l ( θ | χ l ) = ∑ l=1 m ϖ l ∫ h( θ ) d Π l ( θ | χ l ) = ∑ l=1 m ϖ l E Π l ( h( θ ) | χ l ) where E Π l ( h( θ ) | χ l ) = ∫ h( θ ) d Π l ( θ | χ l ) .

Example 3. A mixture of probability distributions (1) is a probability distribution; i.e. ∫ Θ π ( θ | ϖ , χ ) d θ = ∑ l=1 m ϖ l ∫ Θ d π l ( θ | χ l ) d θ ⏟ =1 = ∑ l=1 m ϖ l =1
Definition 4. The mixture is called finite mixture if m< ∞ , and countably infinite mixture if m → ∞ . Here, we focus on finite mixtures.

Definition 5. A mixture model is called is called parametric mixture model if its components are members of the same parametric family of distributions eg. P m ={ Π ( θ | χ l ) ;l=1,...,m} , and hence Π ( θ | ϖ , χ ) = ∑ l=1 m ϖ l Π ( θ | χ l )
Example 6. An example of a parametric mixture model is the Normal mixture model π ( θ | ϖ , μ , σ 2 ) = ∑ l=1 m ϖ l N ( θ | μ l , σ l 2 ) , where all components belong to the Normal distribution family.
Note 7. Mixture distributions are useful because (among others):
  • they can approximate complicate other distributions by using a combination of simpler distributions { Π l ( θ | χ l ) } which lead to more convenient computations.
  • they can naturally model heterogeneity. E.g. consider a population which is heterogeneous in the sense that there are multiple sub-groups labeled by ℓ ∈ {1,...,m} , each group represented in the population with proportion ϖ ℓ , and distributed as Π l ( θ | χ l ) . Then y ∼ Π ( θ | χ ) can be realized by drawing ℓ with probability Pr ( ℓ =l ) = ϖ l and drawing θ from Π ℓ ( θ | χ ℓ ) given ℓ , aka { θ | ℓ ∼ Π ℓ ( θ | χ ℓ ) ℓ ∼ Pr ( ℓ ) which implies  θ ∼ Π ( θ | χ ) since Π ( θ | χ ) = ∑ ℓ =1 m Π ( θ | χ ℓ ) Pr ( ℓ ) = ∑ l=1 m ϖ l Π ( θ | χ l ) See Section 1 in Handout 2.
Example 8. A bimodal, or a right skewed (non-symmetric) distribution can be approximated by a Mixture of (unimodal and symmetric) Normal distributions with different parameter values. Also it describes a population with two groups Normally distributed with different parameters.

2 Mixture prior distributions

Note 9. Mixture models can be used to specify priors distributions, either as a mean to approximate Your actual prior distribution with simpler & tractable distributions, or as a mean to represent heterogeneous prior believes.
Theorem 10. Let y:=( y 1 ,..., y n ) be observables generated from the sampling distribution F( y| θ ) . Prior mixture distribution Π ( θ | ϖ ) is called the prior with pdf/pmf π ( θ | ϖ ) = ∑ l=1 m ϖ l π l ( θ ) , (2) where, P m ={ π l ( θ ) , θ ∈ Θ } l=1 m is a collection of distributions, and { ϖ l } are weights such that ∑ l=1 m ϖ l =1 and ϖ l >0 . Then:
  1. the posterior distribution Π ( θ |y, ϖ ) has pdf/pmf π ( θ |y, ϖ ) = ∑ l=1 m ϖ l * π l ( θ |y ) , (3)
  2. the predictive distribution of a future outcome z has pdf/pmf g( z|y, ϖ ) = ∑ l=1 m ϖ l * g l ( z|y )
where ϖ l * = ϖ l f l ( y ) ∑ l=1 m ϖ l f l ( y ) ∝ ϖ l f l ( y ) f l ( y ) = ∫ Θ f( y| θ ) d Π l ( θ ) π l ( θ |y ) = f( y| θ ) π l ( θ ) ∫ Θ f( y| ϑ ) d Π l ( ϑ ) ∝ f( y| θ ) π l ( θ ) g l ( z|y ) = ∫ Θ f( z| θ ) π l ( θ |y ) d θ
Proof. From Bayes theorem, we have: d Π ( θ |y ) = f( y| θ ) d Π ( θ ) ∫ Θ f( y| ϑ ) d Π ( θ ) = f( y| θ ) ∑ l=1 m ϖ l d Π l ( θ ) ∫ Θ f( y| ϑ ) ∑ l'=1 m ϖ l' d Π l' ( ϑ ) = ∑ l=1 m ϖ l f( y| θ ) d Π l ( θ ) ∑ l'=1 m ∫ Θ ϖ l' f( y| ϑ ) d Π l' ( ϑ ) = ∑ l=1 m ϖ l f l ( y ) f( y| θ ) d Π l ( θ ) f l ( y ) ⏞ = d Π l ( θ |y ) ∑ l'=1 m ∫ Θ ϖ l' f l '( y ) f( y| ϑ ) d Π l' ( ϑ ) f l' ( y ) ⏟ = d Π l' ( ϑ |y ) = ∑ l=1 m ϖ l f l ( y ) d Π l ( θ |y ) ∑ l'=1 m ϖ l' f l' ( y ) ∫ Θ d Π l' ( ϑ |y ) ⏟ =1 = ∑ l=1 m ϖ l f l ( y ) ∑ l'=1 m ϖ l' f l' ( y ) ⏟ = ϖ l * d Π l ( θ |y ) = ∑ l=1 m ϖ l * d Π l ( θ |y ) . Also g( z|y, ϖ ) = ∫ Θ f( z| θ ) d Π l ( θ |y ) = ∫ Θ f( y| θ ) ∑ l=1 m ϖ l * d Π l ( θ |y ) = ∑ l=1 m ϖ l * g l ( z|y )
Remark 11. Theorem 10 shows that the posterior distribution Π ( θ |y ) (derived by a mixture prior) is a mixture of 'individual posterior distributions' Π l ( θ |y ) weighted by ϖ l * . It is determined not only by the observables but also by the weights of the individual distributions. Note that, for l=1,...,m , the prior weights ϖ l and posterior weights ϖ l * my differ a lot, however they can be close each other.

Remark 12. Mixture priors Π ( θ ) whose components { Π l ( θ ) } are conjugate to the likelihood f( y| θ ) can facilitate tractable Bayesian inference. In Theorem 10, if each Π l ( θ ) is conjugate to the likelihood f( y| θ ) , then obviously each Π l ( θ |y ) will belong to the same distribution family as the corresponding Π l ( θ ) because π l ( θ |y ) ∝ f( y| θ ) π l ( θ ) . Then, provided that components π l ( θ ) are tractable, the components π l ( θ |y ) will be tractable too (as they belong to the same distr. family). Likewise, the posterior weights can be calculated in closed form i.e., ϖ l * ∝ ϖ l f l ( y ) since the integral f l ( y ) = ∫ Θ f( y| θ ) d Π l ( θ ) will be tractable. Hence, the produced posterior mixture pdf/pmf will be tractable.

Remark 13. Mixtures of conjugate priors are as easy to manipulate as regular conjugate distributions, while leading to a greater freedom in the modeling of the prior information.
