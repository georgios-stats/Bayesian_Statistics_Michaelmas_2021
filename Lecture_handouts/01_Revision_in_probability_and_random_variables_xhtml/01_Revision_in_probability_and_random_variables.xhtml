1 Linear Algebra

Fact 1. [Cholesky decomposition] Every symmetric positive definite matrix A ∈ R d × R d can be decomposed into a product of a unique lower triangular matrix L ∈ R d × R d and its transpose L ⊤ , i.e. A=L L T Matrix L is called lower triangular factor of the Cholesky decomposition, and it is often denoted as A 1/2 =L .

2 Probability


Definition 2. A collection F={A, A 1 , A 2 ,...} of sets A, A 1 , A 2 ,... , each of which are subsets of set Ω , is called a σ -algebra if and only if
  1. Ω ∈ F
  2. If A ∈ F , then A ∁ ∈ F
  3. If A 1 , A 2 ,... ∈ F is an infinite sequence of sets in F then ∪ i=1 ∞ A i ∈ F

Definition 3. Probability distribution P on ( Ω ,F ) is called a non-negative function if and only if
  1. P( Ω ) =1
  2. If A,B ∈ F and A ∩ B= ∅ then P( A ∪ B ) =P( A ) +P( B )
  3. If A 1 , A 2 ,... ∈ F and A i ∩ A j = ∅ for all i ≠ j then P( ∪ i=1 ∞ A i ) = ∑ i=1 ∞ P( A i )

Definition 4. We call the triple ( Ω ,F,P ) as probability space.

3 Random variables

Definition 5. A d -dimensional random variable y on a probability space ( Ω ,F,P ) is a function y: Ω → R d such as, for any subset A ⊆ R d , it is { ω ∈ Ω :y( ω ) ∈ A} ∈ F .

Definition 6. A d -dimensional random variable y on a probability space ( Ω ,F,P ) , induces a probability P y ( ⋅ ) such that, for all subsets A ⊆ R d , P y ( y ∈ A ) =P( { ω ∈ Ω :y( ω ) ∈ A} ) Essentially, it induces a probability space ( R d , B , P y ) , with B a σ -algebra containing sub-sets of R d .

Definition 7. The (cumulative) distribution function (CDF) of a d -dimensional random variable y ∈ Y is the function F y : R d → [ 0,1 ] such that F y ( y ) := F y ( y ' 1 ,...,y ' d ) = P y (y ∈ (- ∞ ,y ' 1 ] × ... × ( - ∞ ,y ' d ] )
Notation 8. As y ∼ F y , we will denote that the random variable y follows a distribution with distribution function F y . This is because the distribution function defines the distribution of the random variable.

Definition 9. The d -dimensional random variable y: Ω → Y with distribution F y is discrete, if Y is a countable set and the distribution can be described by its Probability Mass Function (PMF) f y ( y' ) := f y ( y ' 1 ,...,y ' d ) =P( { ω ∈ Ω :y( ω ) =y'} )

Definition 10. The d -dimensional random variable y: Ω → Y with distribution F y is absolutely continuous, if Y is a uncountable set and the distribution can be described by its Probability Density Function (PDF) f y ( y ) such that P y ( y ∈ A ) = ∫ ⋯ ∫ ⏟ A f y ( y ' 1 ,..., y d ' ) d y ' 1 ⋯ d y ' d ,  for any  A ⊆ R d . or briefly P y ( A ) = ∫ A f y ( y' ) d y' , where d y'= ∏ j=1 d d y ' j .

Fact 11. The PDF of d -dimensional random variable y: Ω → Y with CDF F y can be computed by the partial derivative as f y ( y ) = . d d t 1 ⋯ d t d F y ( t 1 ,..., t d ) | t 1 = y 1 , ⋯ t d = y d if  F y  is differential.

6 Independence

Definition 15.   Given a probability space ( Ω , B ,P ) , events A,B ∈ B are independent if and only if P( A ∩ B ) =P( A ) P( B )

Fact 16. Let ( n+m ) -dimensional random variable y ∈ Y . Consider a partition y=( x,z ) where x ∈ X is n - dimensional and z ∈ Z is m - dimensional. Let F( ⋅ ) denote CDF, and f( ⋅ ) denote PDF/PMF.
  • The r.v. x and y are independent if and only if, for any A ⊆ X and B ⊆ Z , P( {x ∈ A} ∩ {z ∈ B} ) =P( x ∈ A ) P( z ∈ B )
  • The r.v. x and y are independent if and only if F( x,z ) =F( x ) F( z ) , ∀ x ∈ X , ∀ z ∈ Z
  • The r.v. x and z are independent if and only if f( x,z ) =f( x ) f( z ) , ∀ x ∈ X , ∀ z ∈ Z

9 Characteristic function

Characteristic functions provide an alternative way to the probability function for describing a random variable.
Definition 27.   The characteristic function (CF) of a d dimensional random variable X with distribution F( ⋅ ) is ϕ x ( t ) = E ( e i t T x ) = ∫ e i t T x d F( x ) for t ∈ R d , where e i t T x = cos ( t T x ) +i sin ( t T x ) .
Proposition 28.   Some properties of characteristic functions
  1. ϕ x ( t ) exists for all t ∈ R d and is absolutely continuous
  2. ϕ x ( 0 ) =1 and | ϕ x ( t ) | ≤ 1 for all t ∈ R d
  3. ϕ A+Bx ( t ) = e i t T A ϕ x ( B T t ) if A ∈ R d and B ∈ R k × d are constants
  4. ϕ x+y ( t ) = ϕ x ( t ) ϕ y ( t ) if and only if x and y are independent
  5. if M x ( t ) = E ( e t T x ) is the moment generating function, then M x ( t ) = ϕ x ( -it )

Conditional independence

Definition 37. Given a probability space ( Ω , B ,P ) , and events A,B,C ∈ B , A and B are conditionally independent given C if and only if P( A ∩ B|C ) =P( A|C ) P( B|C ) , for P( C ) >0.

Fact 38. Let ( n+m+d ) -dimensional random variable y ∈ Y ⊆ R n+m+d . Consider a partition y=( x,z, θ ) where x ∈ X ⊆ R n , z ∈ Z ⊆ R m , and θ ∈ Θ ⊆ R d .
  • The r.v. x and y are independent given θ if and only if F( y,z| θ ) =F( y| θ ) F( z| θ ) , ∀ x ∈ R n , ∀ z ∈ R m where F( ⋅ | θ ) denotes the conditional CDF.
  • This implies that r.v. x and z are independent given θ if and only if f( y,z| θ ) =f( y| θ ) f( z| θ ) where f( ⋅ | θ ) denotes the conditional PDF/PMF.

This offers a probabilistic mechanism for (1.) inversion , or (2.) updating .
Fact 39. [Bayesian theorem with sets] Assume a probability space ( Ω ,F,P ) . For any sets A,B ∈ F , it is P( A|B ) = P( B|A ) P( A ) P( B ) , provided that  P( B ) ≠ 0 .
