1 Conjugate priors

Note 1. We aim at specifying a prior distribution family, which (i.) leads to a tractable (to some extend) posterior distribution, (ii.) is rich enough to allow us to quantify prior believe, and (iii.) has a reasonable interpretation.
Definition 2. Let F ={F( y| θ ) ; ∀ θ ∈ Θ } be a family of sampling distributions. A family of prior distributions P on Θ is said to be (natural) conjugate for F if the posterior Π ( θ |y ) belongs to P for all prior Π ( θ ) ∈ P and all F( y| θ ) ∈ F ; i.e. Π ( θ |y ) ∈ P , ∀ F( y| θ ) ∈ F and Π ( θ ) ∈ P .
Note 3. By specifying a tractable conjugate prior distribution Π ( θ ) , we can achieve tractability for the posterior Π ( θ |y ) since it belongs to the same distribution family as the prior.

1.1 General derivation

We restrict here the derivation of conjugate priors in statistical models satisfying the following assumptions:
Assumption 4. Let y=( y 1 ,..., y n ) be observables. We assume:
  1. the likelihood is such that L( θ | y 1:n ) =L( θ | y 1:k ) L( θ | y k+1:n ) for any k ∈ {1,...,n-1} , and
  2. there exists a parameteric sufficient statistic t: Y → R k with t( y 1 ,..., y n ) =t ∈ R k where its dimension k is independent on the number of observables n .
Lemma 5. Let t ( 1 ) =t( y 1 ,..., y q ) and t ( 2 ) =t( y q+1 ,..., y n ) be sufficient statistics of two data-sets. Then, under the conditions of Assumption 4, there exists a binary operator * such that y ( 1 ) * y ( 2 ) = y * :=( y 1 * ,..., y k * ) (1) such that f( y 1 ,..., y n | θ ) ∝ k( y * | θ ) and k( y * | θ ) ∝ k( y ( 1 ) | θ ) k( y ( 2 ) | θ )
 
Note 6. Let statistic t: Y → R k with t( y 1 ,..., y n ) =t ∈ R k be parametric sufficient, and let its dimension k be independent from data size n . Then from Neyman factorization theorem, the likelihood can be factorized as f( y| θ ) =k( t( y ) | θ ) ρ ( y ) ∝ k( t( y ) | θ ) , (2) where ρ ( y ) is the residual term of a likelihood kernel k( t( y ) | θ ) of θ . Note 7. Essentially the prior Π ( θ ) in (4) and the posterior in Π ( θ |y ) (5) belong to the same distribution family P due to (5). The only difference is in the hyper-parameter values which change from prior to posterior as . In the posterior distribution the hyper-parameters combine both the prior info quantified in τ and the experimental info quantified in t( y ) according to the binary operator * .
 
Theorem 8. Let y=( y 1 ,..., y n ) be observable quantities drawn from F( y| θ ) independently conditional on θ , and let f( y| θ ) be the likelihood with sufficient statistic t:=t( y ) of a fixed dimension k independent from n . The conjugate prior Π ( θ ) with hyper-parameter τ of the likelihood f( y| θ ) can be specified by setting its pdf/pmf as π ( θ ) := π ˜ ( θ | τ ) = 1 N( τ ) k( τ | θ ) ∝ k( τ | θ ) (6) where k( ⋅ | θ ) is a kernel of the likelihood from the Neyman factorization f( y| θ ) =k( t( y ) | θ ) ρ ( y ) ∝ k( t( y ) | θ ) , and τ are hyper-parameters such that N( τ ) = ∫ k( τ | θ ) d θ < ∞ .

Note 9. Essentially, in Theorem 8, and Note 6, we expect that since the likelihood kernel k( τ | θ ) is tractable, it may lead to a tractable conjugate prior, and hence to a tractable posterior distribution.

Note 10. Once the conjugate family of prior distributions Π ( θ ) has been specified, You can assign values on the hyper-parameters τ based on your a priori information. In fact, the values assigned on τ do not necessarily need to lie in the support of the sufficient statistics T ; the only restriction is that τ has to lead to a proper posterior N( t( y ) * τ ) < ∞ .

Note 13. Exponential family of distributions cover a large range of distributions satisfying the conditions in Note 4.

Fact 14. (Pitman-Koopman-Lemma) If a distribution family {F( y| θ ) , ∀ θ ∈ Θ } is such that there exists a sufficient statistic whose dimension is independent on the number of observations and the support y ∈ Y of F( y| θ ) does not depend on θ , then it is an exponential family.
Note 15. When the parametric model is member of the Exponential family, a conjugate prior distribution on its uncertain parameters can be specified as shown in Theorem 16.

Note 17. 2 Conditional conjugate priors

Note 21. In some problems involving more realistic/complicated statistical models, certain computational tools, (e.g., the Gibbs sampler in Term 2), require the availability of tractable posterior conditionals instead of that of the full joint posterior. Specifying, conditional conjugate priors is a way to achieve this.
Definition 22. Let F ={F( y| θ 1 , θ 2 ) ; ∀ θ 1 ∈ Θ 1 , ∀ θ 2 ∈ Θ 2 } be a family of sampling distributions. A family of prior distributions P θ 1 for θ 2 conditional on θ 1 is said to be conditional conjugate for F if the posterior Π ( θ 2 |y, θ 1 ) belongs to P θ 1 for all prior Π ( θ 2 | θ 1 ) ∈ P θ 1 and all F( y| θ 1 , θ 2 ) ∈ F ; i.e. Π ( θ 2 |y, θ 1 ) ∈ P θ 1 , ∀ F( y| θ 1 , θ 2 ) ∈ F and Π ( θ 2 | θ 1 ) ∈ P θ 1 .

Note 23. The conditional conjugate prior for θ 2 conditional θ 1 is specified from Theorem 8 as the conjugate prior of θ 2 on F( y| θ 1 , θ 2 ) given that parameter θ 1 is known/fixed. Based on this, Neyman factorization is applied as f( y| θ 1 , θ 2 ) = k 1 ( t( y ) | θ 1 ) ρ ( y| θ 1 ) ∝ k( t( y ) | θ 1 ) , and the prior is specified according to π ( θ 1 ) ∝ k 1 ( τ 1 | θ 1 ) (9) for some researchers specified prior hyper-parameter vector τ 1 . Likewise, I get the conditional conjugate prior for θ 1 conditional θ 2 as π ( θ 2 ) ∝ k 2 ( τ 2 | θ 2 ) . The join prior Π ( θ ) satisfying conditional conjugation for θ 1 and θ 2 is π ( θ ) = π ( θ 1 ) π ( θ 2 ) ∝ k 1 ( τ 1 | θ 1 ) k 2 ( τ 2 | θ 2 ) . This derivation is extendable to any number of blocks θ =( θ 1 ,..., θ B ) .
