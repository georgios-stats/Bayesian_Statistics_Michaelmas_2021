Exercise 2. ( ★ ★ )
Suppose we wish to estimate the values of a collection of discrete random variables X → = X 1 , …, X n . We have a posterior joint probability mass function for these variables, p( x → | y ) =p( x 1 , … , x n | y ) based on some data y . We decide to use the following loss function: ℓ ( x → ˆ , x → ) = ∑ i=1 n ( 1- δ ( x ˆ i , x i ) ) (1) where δ ( a,b ) =1 if a=b and zero otherwise.
  1. Derive an expression for the estimated values, found by minimizing the expectation of the loss function. [Hint: use linearity of expectation.]
  2. When the probability distribution is a posterior distribution in some problem, this type of estimate is sometimes called `maximum posterior marginal' (MPM) estimate. Explain why this name is appropriate.
  3. Explain in words what the loss function is measuring. Compare with the loss function for MAP estimation.
Solution.  
  1. We have that E( ℓ ( x → ˆ , X → ) | y ) =E( 1- ∑ i=1 n δ ( x ˆ i , X i ) | y ) (2) =n- ∑ i=1 n E( δ ( x ˆ i , X i ) | y ) =n- ∑ i=1 n ∑ x i ∈ X i δ ( x ˆ i , x i ) p( x i | y ) =n- ∑ i=1 n p( x ˆ i | y ) (3) To minimize this, it suffices to minimize each term of the sum separately, and so we have for each i ∈ {1, … ,n} separately: x ˆ i * = arg max x i ∈ X i p( x i | y ) (4) [The x i in (4) corresponds to the x ˆ i in ; we simply drop the hat to keep notation as simple as possible.]
    Contrast this with the MAP estimate, which requires optimisation over the full joint pmf: x → ˆ * = arg max x → ∈ X → p( x ˆ i | y ) (5)
  2. The individual terms of the sum in (3) are the posterior marginal distributions for each x i found from the joint distribution p( x ˆ i | y ) . The estimate for each x i is found by maximizing its own posterior marginal distribution, hence the name.
  3. MPM estimation has attractive properties. Like MAP estimation, it can be defined on any set (albeit with the same caveats as for MAP with continuous variables). On the other hand, it does not insist that all variables `match' to get the minimum loss. Rather, it counts the number that match by summing over the individual zero-one losses. The loss function for MAP estimation, on the other hand, imposes a different penalization since it is ℓ ( x → ˆ , x → ) =1- δ ( x → ˆ , x → ) =1- ∏ i=1 n δ ( x ˆ i , x i ) (6) where by taking the product, the loss will be one unless all variables match.
