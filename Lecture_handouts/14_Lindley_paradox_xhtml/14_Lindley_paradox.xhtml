1 Improper priors situations

Jeffreys-Lindley Paradox describes the overwhelming statistical evidence in favor a hypothesis when the priors of the alternative hypotheses diverge faster. It occurs in Bayesian hypothesis test and model selection with improper priors.
Example 1. [Single-vs-General-alternative]
Let y=( y 1 ,..., y n ) observables and consider the Bayesian hypothesis test H 0 : y i | θ 0 ∼ IID N ( θ 0 , σ 2 ) ,i=1,...,n vs H 1 :{ y i | μ ∼ IID N ( μ , σ 2 ) ,i=1,...,n μ ∼ N ( μ 0 , σ 0 2 ) (1) with π j = Pr Π ( θ ∈ Θ j ) for j=0,1 . Let θ 0 , σ 2 , μ 0 , σ 0 2 be fixed values. It can be computed (Appendix A) that B 01 ( y ) = ( σ 2 n ) - 1 2 ( σ 2 n + σ 0 2 ) - 1 2 exp ( - 1 2 ( y ‾ - θ 0 ) 2 σ 2 n ) exp ( - 1 2 ( y ‾ - μ 0 ) 2 σ 2 n + σ 0 2 ) ; Pr Π ( H 0 |y ) = ( 1+ 1- π 0 π 0 ( σ 2 n + σ 0 2 ) - 1 2 ( σ 2 n ) - 1 2 exp ( - 1 2 ( y ‾ - μ 0 ) 2 σ 2 n + σ 0 2 ) exp ( - 1 2 ( y ‾ - θ 0 ) 2 σ 2 n ) ) -1
  1. The Jeffreys' (and Laplace) prior of μ is π ( J ) ( μ ) ∝ 1 ( μ ∈ R ) (see Handout 7). Find values μ * , σ * 2 such that π ˜ ( μ | H 1 ) → π ˜ ( J ) ( μ ) ; as ( μ 0 , σ 0 2 ) → ( μ * , σ * 2 ) (2) where π ˜ ( μ | H 1 ) denotes the kernel of the pdf of the conditional prior μ ∼ N ( μ 0 , σ 0 2 ) of μ under H 1 in (1), and π ˜ ( J ) ( μ ) denotes that of the the Jeffreys prior π ( J ) ( μ ) ∝ 1 ( μ ∈ R ) .
  2. [Lindley's Paradox] Let σ 0 2 → ∞ . Investigate, how, and why
    • the conditional prior π ( μ | H 1 ) given the hypothesis H 1 behaves?
    • the Bayes Factor B 01 ( y ) and the posterior Pr Π ( H 0 |y ) behave?
Solution. The calculation of B 01 ( y ) and of Pr Π ( H 0 |y ) is presented in the Appendix A.
  1. The kernel π ˜ ( μ | H 1 ) of the prior π ( μ | H 1 ) is π ( μ | H 1 ) = N ( μ | μ 0 , σ 0 2 ) ∝ exp ( - 1 2 1 σ 0 2 ( μ - μ 0 ) 2 ) 1 ( μ ∈ R ) = π ˜ ( μ | H 1 ) So for σ * 2 = ∞ and any | μ * |< ∞ , I get the (2).
  2. If σ 0 2 → ∞ , the prior N ( μ 0 , σ 0 2 ) of H 1 becomes flat (the mass spreads uniformly around R ) and improper: π ( μ | H 1 ) = N ( μ | μ 0 , σ 0 2 ) ∝ exp ( - 1 2 ( μ - μ 0 ) 2 σ 0 2 ) 1 ( μ ∈ R ) → σ 0 2 → ∞ 1 ( μ ∈ R ) In fact π ( μ | H 1 ) meets Jeffreys' (or Laplace) prior of μ in the limit σ 0 2 → ∞ .
    Then in the limit the Bayes factor B 01 ( y ) and the posterior Pr Π ( H 0 |y ) approach the values B 01 ( y ) → ∞ , and Pr Π ( H 0 |y ) →1  as  σ 0 2 → ∞ We observe that regardless the number of the observables n , when the prior variance σ 0 2 of μ given H 1 becomes huge ( σ 0 2 → ∞ ) , the prior of μ in H 1 becomes more diverge π ( μ | H 1 ) → σ 0 2 → ∞ 1 ( μ ∈ R ) ,  as  σ 0 2 → ∞ spreading the prior mass uniformly in R while the evidence in favor H 0 : μ = θ 0 that μ is equal to single value θ 0 becomes overwhelming. B 01 ( y ) → σ 0 2 → ∞ ∞ and Pr Π ( H 0 |y ) → σ 0 2 → ∞ 1 Paradoxical behavior: One would not expect for σ 0 2 → ∞ favor H 0 : μ = θ 0 because σ 0 2 → ∞ gives increases ignorance and give uniformly positive mass to more values. Also, we wouldn't expect increasing the sample size n to have no effect.
    Essentially, the above paradoxical behavior says that: When σ 0 2 → ∞ , it means that a priori, I know μ = θ 0 with probability π 0 , but I know nothing about μ probability 1- π 0 ; however after I get any observation y I am a posteriori certain that μ = θ 0 .
    Observe that the overall posterior is π ( μ ) = π 0 1 ( μ ∈ { θ 0 } ) +( 1- π 0 ) [ ( 1 2 π σ 0 2 ) 1 2 exp ( - 1 2 ( μ - μ 0 ) 2 σ 0 2 ) 1 ( μ ∈ R ) ] When σ 0 2 → ∞ , it involves one conditional component that concentrates the mass π ( μ | H 0 ) = 1 { θ 0 } ( μ ) and one component with infinite normalizing constant. So we cannot apply the Bayes theorem by using the kernels of the priors and in the sense of canceling normalizing constants as π ( μ |y ) ∝ f( y| μ ) π ˜ ( μ ) where π ( μ ) ∝ π ˜ ( μ ) .

Lindley's paradox also appears in composite-vs-composite tests with improper priors, when the one prior diverges faster than the other in the limit. See the following realistic example in Bayesian regression Variable selection problem.
